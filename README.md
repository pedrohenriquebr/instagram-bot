# Instagram Bot

## Settings

Create a file named .env based on .e
Fill out the .env with vars

```dotenv
USERNAME=username
PASSWORD=password
LINK=post link
CUSTOM_COMMENT="I will win, I want"
# Max and min of random delay seconds 
# between each comment
MIN_RANDOM_DELAY=60
MAX_RANDOM_DELAY=120
HEADLESS=True
GECKODRIVER="geckodriver path"
FIREFOX_PATH="firefox path"
COMMENTS_LIMIT=10
```


## Features

* Login
* Comment
* Metion 2 accounts
* Comments Limit
* Open post link
* Scrape Followers
* Local accounts file
